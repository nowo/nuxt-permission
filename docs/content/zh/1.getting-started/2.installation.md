## 安装

以下两种方式**二选一**。

### 方式一 —— Nuxt CLI（推荐）

`nuxt module add` 会一步到位:安装依赖**并**自动把模块注册进 `nuxt.config.ts`:

```bash
npx nuxt module add nuxt-permission
```

### 方式二 —— 手动安装

用你的包管理器安装:

::code-group
```bash [pnpm]
pnpm add -D nuxt-permission
```

```bash [npm]
npm install -D nuxt-permission
```

```bash [yarn]
yarn add -D nuxt-permission
```

```bash [bun]
bun add -D nuxt-permission
```
::

然后把它加进 `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
    modules: ['nuxt-permission'],
})
```

## Install

Pick **one** of the two approaches below.

### Option 1 — Nuxt CLI (recommended)

`nuxt module add` installs the package **and** registers it in `nuxt.config.ts` for you, in one step:

```bash
npx nuxt module add nuxt-permission
```

### Option 2 — Manual

Install with your package manager:

::code-group
```bash [pnpm]
pnpm add -D nuxt-permission
```

```bash [npm]
npm install -D nuxt-permission
```

```bash [yarn]
yarn add -D nuxt-permission
```

```bash [bun]
bun add -D nuxt-permission
```
::

Then register it in `nuxt.config.ts`:

```ts [nuxt.config.ts]
export default defineNuxtConfig({
    modules: ['nuxt-permission'],
})
```
